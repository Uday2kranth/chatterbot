            Welcome! This guide is designed specifically for complete beginners who want to set up API keys for various AI model providers. All you need to follow these steps is a Google Account or a GitHub Account.
          


          
          

            

              
              ⚠️ IMPORTANT ROUTING DISCLAIMER: BACKGROUND WHISPER USAGE
            

            

              No matter which operator or model is active (e.g. Mistral, Gemini, or NIM), clicking the Record or Audio upload buttons will **always route through the Groq Whisper API** in the background for transcribing.
            

            

              💡 Warning / Strategy: Because your Groq API key limits are vital for this audio-to-text transcription service, try to avoid using Groq for regular chat/thinking models when other free operators (like Gemini, OpenRouter, or NVIDIA) are available. This will conserve your Groq key's daily rate budget exclusively for voice and audio features!
            

          


          
          

            

              

                
                🚨 CRITICAL WARNINGS FOR BEGINNERS
              

              
              

                

                  1. The "Login Redirect" Scenario

                  When you click any direct link below to create an API key:
                  
    
                    
  • If you are already logged in: The link will take you directly to the key generation page.
    • 
                    
  • If you are NOT logged in: You will be redirected to their Login/Sign-Up page first.
    • 
                  

                  ⚠️ What to do: After logging in with Google/GitHub, if you get lost on their generic homepage, simply come back here and click the link again. Since you are now logged in, it will take you straight to the correct page.
                

                
                

                
                

                  2. The "One-Time Show" Rule (IMPORTANT)

                  For almost all of these providers, your secret API key is only visible ONCE when you first generate it.

                  ⚠️ What to do: Copy the key immediately and save it securely in a text file or paste it into our Settings panel. If you refresh or click away, you will lose access forever and will need to delete it and create a new one.
